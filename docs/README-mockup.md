Mockup: Estudio Socioeconómico

Descripción

Este mockup es una página A4 de ejemplo para el formulario "Estudio Socioeconómico". Está pensado para servir como punto de partida visual: logo en encabezado, recuadro `FOTO (1)` en la esquina superior izquierda, secciones separadas por tarjetas y campos para impresión manual o llenado digital.

Archivos

- [estudio-socioeconomico-mockup.html](estudio-socioeconomico-mockup.html)
- [estudio-socioeconomico.css](estudio-socioeconomico.css)

Instrucciones rápidas

1. Abrir `docs/estudio-socioeconomico-mockup.html` en el navegador (arrastra el archivo al navegador o usa `File > Open`).
2. Para usar el logo institucional, coloca tu `marakame.jpeg` en `frontend/src/assets/` (ya existe) o edita la ruta del `img` en el HTML.
3. Para exportar a PDF desde el navegador: `Ctrl+P` (Windows) → Configurar destino "Guardar como PDF" → Ajustar tamaño a "A4" y márgenes "Por defecto" → "Guardar".

Personalización sugerida

- Cambiar colores en `estudio-socioeconomico.css` (variable `#7E1D3B` se usa como color principal). 
- Ajustar tipografías: agregar `@import` de Google Fonts y usar `font-family` deseada.
- Reemplazar el recuadro `FOTO (1)` por una `img` con la foto del paciente y ajustar tamaño a 6x6 cm.
